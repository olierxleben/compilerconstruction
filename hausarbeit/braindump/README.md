### Optmierungsregeln

* sharthand (Beispiel: margin-top: 50px; margin-left:50px; => margin: 50px 0 0 50px;)

* Linebreaks-Reduzierung
* Entfernung unnötiger Zeichen (zum Beispiel letztes Semikolon)
* Farb-Shorthands
* Entfernen von "px"
* Entfernen von unbenutzten Selektoren
* Kompression
* Strukturierung
